Applying for API Keys

Twitter API

Apply for Twitter API keys by visiting Twitter's devoloper portal and clicking 'apply' in the top left corner. One must compleate a survey about their use with the API. Be sure to read up on restricted use cases before applying. Twitter restricts some automated processes (for examples, bots are not allowed to like tweets), despite the API accomidating these tasks. It's important to reign our ideas within Twitter's guidelines so that we don't lose our API access!

Google Drive/Sheets API

Here's Google's devoloper section for the Drive API. One can apply and view their API keys and projects in the devoloper console. Once approved, under the credentials tab, your authentication info will be available for download. Store these files in the same folder as any scripts requesting access to the Sheets API.
